# Personal-Portfolio-website
This is a responsive personal portfolio website built with HTML, CSS, JavaScript and React. It showcases my work, skills, projects, and contact information in a clean and modern layout. 
Features

-  About Me section
-  Skills showcase
-  Projects listing
-  Contact form or contact info
-  Smooth animations
-  Responsive design

Technologies Used

- **React.js** – component-based architecture
- **HTML5** – semantic markup
- **CSS3** – custom styles and responsiveness
- **JavaScript (ES6+)** – logic and interactivity
